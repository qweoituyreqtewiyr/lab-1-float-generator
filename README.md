# Генератор вещественных чисел

Приложение на языке C для практического задания N 1 по теме
«Машинное представление вещественных чисел».

Программа генерирует индивидуальные варианты заданий с вещественными
числами и формирует файлы в формате Markdown.

## Функциональность

- чтение параметров генерации из файла `input.txt`;
- генерация заданного количества вариантов;
- генерация `k` вещественных чисел для каждого варианта;
- поддержка разрядности 32, 64 и 128 бит;
- перевод чисел в машинное представление;
- расчет точности представления числа;
- создание Markdown-файлов с заданиями и ответами.

## Входные данные

Параметры задаются в файле `input.txt`:

```txt
n
k
bits
a
b
p
```

Где:

- `n` - количество вариантов;
- `k` - количество заданий для одного варианта;
- `bits` - разрядность вещественного числа: 32, 64 или 128;
- `a`, `b` - границы диапазона генерации чисел;
- `p` - количество знаков после запятой.

Пример:

```txt
3
20
128
-10
10
2
```

## Выходные данные

После запуска программа создает два типа файлов:

- `Задания/variant_N.md` - таблица с номерами и вещественными числами;
- `Проверка/variant_N.md` - таблица с числами, машинным представлением и точностью.

## Структура проекта

```txt
.
├── main.c
├── input.txt
├── Задания/
├── Проверка/
└── README.md
```

## Сборка и запуск

Сборка через `gcc`:

```bash
gcc main.c -o generator -lm
```

Запуск:

```bash
./generator
```

После запуска результаты будут записаны в каталоги `Задания` и `Проверка`.

## Используемые технологии

- язык программирования C;
- стандартные библиотеки C: `stdio.h`, `stdlib.h`, `math.h`, `string.h`, `time.h`;
- Git для контроля версий.
